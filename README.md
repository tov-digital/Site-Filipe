# Filipe Carvalho Advocacia

Este é o repositório do site oficial para **Filipe Carvalho Advocacia**. O site é uma aplicação web estática focada na apresentação dos serviços jurídicos oferecidos pelo escritório.

## 🚀 Como Executar Localmente

Para visualizar o projeto em sua máquina local, você pode seguir os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/tov-digital/Site-Filipe.git
    cd Site-Filipe
    ```

2.  **Instale as dependências (opcional):**
    Este projeto utiliza o pacote `serve` para rodar localmente.
    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
    O site estará disponível em `http://localhost:3000`.

## 🛠️ Tecnologias Utilizadas

*   **HTML5 / CSS3 / JavaScript** (Vanilla)
*   **Font Awesome** para ícones.
*   **Google Fonts** (Lora, Sora, Ubuntu).
